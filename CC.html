Curriculum Connection

Website for PowerSchool Unniversity Events

Designed and developed web based course catalog for PSU Events called Curriculum Connection from scratch. This website was designed to train the customers on different PowerSchool products. The effort was to make it intutive for them. This website is responsive across monitor, laptop and tablet devices.

I collected several customer and trainer feedback, and decided to re-design the website to enhance the user experience. I reconstructed the information architecture, upgraded the site navigation for easy resource access, introduced an intuitive search function, and added 3 new screens to simplify the user experience. Further, I designed an entirely new module to allow the users to personalize the configurations of their devices.
